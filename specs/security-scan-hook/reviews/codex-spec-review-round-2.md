Fix: either add pre/post Bash state sufficient to include paths changed between the invocation's starting and ending commits (and test the create-or-modify-plus-commit case), or explicitly narrow the Bash guarantee throughout spec.md, decisions.md, Tasks 3-4, and AC4 to files visible as dirty/untracked after a Bash call.
